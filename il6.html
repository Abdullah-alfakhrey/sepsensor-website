Why IL-6

IL-6 first. Platform second.

A tighter wedge beats a vague platform story.

  • First biomarker: IL-6. The opening assay is specific by design.
  • Chosen for sepsis relevance. Project materials consistently position IL-6 around severity and outcome insight.
  • Expansion comes later. Validate the first assay path before broader biomarker growth.

Why it works for the story

One assay. Cleaner validation.

  • Sharper clinical signal. A focused readout is easier to explain and test.
  • Sharper technical path. Reader, cartridge, optics, and software can be validated around one use case first.
  • Sharper commercial wedge. IL-6 first is easier to position than a generic “future platform.”

Assay focus

Prototype development stays specific.

  • Human plasma sample. The source documents frame IL-6 sensing around plasma-based measurement.
  • Fluorescent immunoassay. The assay path is built around fluorescence capture and software interpretation.

Validation logic

One marker helps the hardware learn faster.

  • Reader validation. Camera, optics, and signal processing all benefit from a single first benchmark.
  • Cartridge validation. Flow, surfaces, and handling can be tuned around one assay path.

Future direction

Start narrow. Expand intelligently.

  • Platform direction remains open. Later biomarker expansion is part of the longer-term story.
  • Prototype language stays honest. This page describes the first assay focus, not finished clinical claims.
